# urlshortner-frontend
Front End Code for Url Shortner 

## Development

```bash
npm install
npm run dev      # start the dev server (proxies /api to localhost:8080)
npm test         # run the Vitest suite
npm run build    # type-check + production build
npm run lint     # oxlint
```

See `docs/SETUP.md` for the full setup guide, including the SDLC orchestration engine
in `orchestrator/`.
